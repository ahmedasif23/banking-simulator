# banking-simulator
This project offers a comprehensive command-line banking simulator implemented in C++. It enables users to simulate fundamental banking operations such as depositing and withdrawing money, transferring funds between accounts, checking balances, viewing transaction histories, requesting loans, performing currency calculations, managing payments, and buying cryptocurrencies. The simulator provides a practical environment for learning and experimenting with basic financial transactions through an intuitive command-line interface. Whether you're a student exploring C++ programming or an enthusiast interested in financial simulations, this project serves as an educational tool to understand essential banking functionalities. Get started by cloning the repository and following the setup instructions in the README to run the simulator locally.
